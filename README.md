# 🛒 CartShare

CartShare is a real-time collaborative shopping cart and shopping list application that allows multiple users to create or join a room and manage shopping items together. Users can add items, specify quantity and category, track activities, and generate a shared receipt.

## 📌 Problem Statement

When friends, roommates, or family members shop together, it becomes difficult to maintain a shared shopping list and keep track of who added which items. CartShare solves this problem by providing a real-time collaborative platform where multiple users can manage a shopping list in a shared room.

## 🚀 Features

* Create and Join Shopping Rooms
* Real-Time Collaboration using Firebase Firestore
* Multi-User Shared Shopping List
* Add Items with Quantity and Category
* Activity Log Tracking
* Shopping Statistics Dashboard
* Dark Mode Support
* Generate Shopping Receipt
* Print Receipt
* Responsive Design

## ⚙️ Installation

1. Clone the repository

git clone <repository-link>

2. Open the project in VS Code

3. Configure Firebase

* Create a Firebase project
* Enable Firestore Database
* Enable Authentication
* Replace Firebase configuration in firebase.js

4. Run using Live Server

## 📋 How It Works

1. User enters a name.
2. Create a room or join an existing room.
3. Add shopping items with quantity and category.
4. All room members see updates in real time.
5. Activity log records actions.
6. Generate and print receipt.
